LibBot mendukung integrasi langsung ke **OpenAI API (GPT-4o-mini)** atau **Google Gemini API**. Jika kosong, sistem otomatis menggunakan asisten simulasi lokal.

Data kunci hanya disimpan secara lokal di browser Anda (*localStorage*) dan tidak pernah dikirimkan ke pihak ketiga selain penyedia API terkait.
